# Hydronautica – Next.js + Tailwind Site

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open your browser at http://localhost:3000

This project uses:

- Next.js (app directory)
- Tailwind CSS (dark tech theme)
- Pages:
  - `/` – Landing page
  - `/about`
  - `/openfastr`
  - `/hdmdc`
  - `/coupled-cfd-fea`
